# Images — NurSon

Ce dossier doit contenir vos photos produit.

## Fichiers attendus

| Fichier | Usage | Dimensions recommandées |
|---|---|---|
| `product-1.jpg` | Vue principale produit | 800×800px minimum |
| `product-2.jpg` | Vue de face | 800×800px |
| `product-3.jpg` | Vue de côté / détail | 800×800px |
| `product-4.jpg` | Photo lifestyle / ambiance | 800×800px |
| `og-image.jpg` | Open Graph (réseaux sociaux) | 1200×630px |
| `product-og.jpg` | Open Graph page produit | 1200×630px |
| `hero-bg.jpg` | Fond hero (optionnel) | 1920×1080px |

## Conseils photos

- Fond blanc ou fond épuré pour les vues produit
- Une photo lifestyle dans un intérieur islamique épuré (tapis de prière, plante verte, lumière naturelle)
- Format JPEG optimisé (qualité 80-85%) ou WebP pour les performances
- Nommage cohérent avec le code HTML
